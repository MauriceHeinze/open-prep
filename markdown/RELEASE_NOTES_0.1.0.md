# OpenPrep 0.1.0

OpenPrep 0.1.0 is the first open-source preview of the app: a focused macOS desktop workflow for TOEFL-style writing practice with structured AI feedback.

For a visual release overview, see [RELEASE_README_0.1.0.md](RELEASE_README_0.1.0.md).

## Highlights

- Prompt catalog for choosing a writing task before you start.
- Distraction-free submission flow for writing and sending your response.
- Codex-backed evaluation pipeline for structured writing feedback.
- Feedback screens that surface scores, criterion notes, improvement points, and highlighted phrase-level feedback.
- Local persistence so attempts can be reviewed later.
- Native Electron desktop packaging with a macOS-first experience.

## Notes

- This is an early preview release, so the scope is intentionally narrow.
- The current build focuses on TOEFL-style Writing and does not yet cover the other exam sections.
- AI evaluation depends on the configured local provider path and a working Codex CLI setup.

## Recommended For

- Learners who already use ChatGPT or Codex and want a focused writing practice workflow.
- Anyone testing the current foundation of the desktop app before broader exam support is added.
